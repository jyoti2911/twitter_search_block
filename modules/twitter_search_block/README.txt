 INTRODUCTION
 ------------
The module provides a block that shows the embedded timeline with given Hastag.
The module provides "Hashtag" as a field that can be linked to content type.The block with embedded tweets with Hashtag will appear in content accordingly.
The module includes Twitter API Library.
The module includes easy OAuth Configuration.


REQUIREMENTS
------------
The module requires two modules:
-block
-field

INSTALLATION
------------
 * Install as you would normally install a contributed drupal module. See:
  https://drupal.org/documentation/install/modules-themes/modules-7
  for further information.
